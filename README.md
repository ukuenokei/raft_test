## 目標内容

- AppendEntries を実装
  - リーダーは複数のサーバーにエントリーを投げる
  - フォロワーは受け取ったら成功レスポンスを投げる
  - 投げ合う内容はすべて 0 埋めされた構造体

## 前提条件

- サーバーの状態と情報は servers という配列で管理する
- リーダーとフォロワーをノードで固定する
  - ノード番号とノード数を引数で渡し、ノード番号 0 のものはリーダーになる
- フォロワーはリーダーからハートビートを受け取ったら、成功応答を返す
- タームを 0 で固定する
